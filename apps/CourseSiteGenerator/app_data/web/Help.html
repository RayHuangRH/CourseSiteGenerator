Help

Course Site Generator is a desktop Java application for creating and updating course Web sites. The app presents five tabs. The first tab, site, is for general information for the entire site such as images, banners, pages to include, and instructor information. The second tab, syllabus, provides text areas for the user to enter additional information about the course at hand. The third tab, meeting times, contains three tables, one for lectures, recitations and labs. The fourth tab, office hours, contains two tables, one for the Teaching Assistants in the course and another for their corresponding office hours. Lastly, the tab schedule, contains a table for events for the course throughout a set time period and the ability to add/remove events. The application saves data to a JSON format that can be used to create Web pages.

Step 1: Course Information

Click the drop down arrow in the combo boxes and select the desired information and type a desired title into the title textfield.

Step 2: Include webpages

Click the boxes for desired pages to include.

Step 3: Select images

Click the buttons under the style box to select images to use in the site. Use the combobox to choose a style sheet.

Step 4: Instructor information

Enter instructor information into the textfields

Step 5: Syllabus information

Enter information into the appropriate textareas under eachs section.

Step 6: Meeting Times

Click the + button to add lectures, recitations or labs. Click the - button to remove.

Step 7: Add TA and Hours

Add a TA to the table using the name and email text field. Add a TA to the office hours table by clicking a TA and then clicking the hours desired

Step 8: Remove and Edit TA

Double click a TA to edit their information and click the - button to remove a TA.

Step 9: Schedule

Select a time range on the top and use the add/edit feature on the button to add new items or edit existing items.

Once you've completed your work, you can press the save button and give your file a name. Note that the name in the app text field will be the name used in the Recent Work list of the Welcome dialog.
